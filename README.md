# Nitons Software Archive
### Introduction

This is my Debian/Ubuntu software repository! This repository contains my software packaged for Debian and Ubuntu systems.

### Installation

To install your software from this PPA, follow these steps:

 1. Open a terminal window.
 2. Run the following command to import the signing key:

``` shell
curl -s --compressed "https://archive.niton.dev/KEY.gpg" | sudo apt-key add -

```
